# 🎬 Movie Watchlist Web Application

## 📌 Project Overview

The Movie Watchlist Web Application is an interactive web app that allows users to search for movies and manage a personalized watchlist. It demonstrates the use of JavaScript, API integration, and dynamic UI rendering.

---

## 🎯 Purpose

The purpose of this project is to apply core web development concepts by building a real-world application that:

* Fetches and displays movie data from an external API
* Allows users to interact with the data
* Provides a clean and responsive user interface

---

## 🔗 API Used

This project uses the **OMDb API** to fetch movie data.

API Link: https://www.omdbapi.com/

---

## ✨ Features

### 🔍 Live Movie Search

* Users can search for movies by title
* Results are fetched dynamically from the API

### ❤️ Watchlist Management

* Add movies to a personal watchlist
* Remove movies from the watchlist
* Watchlist persists using local storage

### 📭 Empty States

* Displays a message when no movies are found
* Displays a message when the watchlist is empty

### 🎨 User Interface

* Dark-themed UI inspired by modern streaming platforms
* Responsive design for mobile, tablet, and desktop
* Sticky sidebar to display saved movies

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Fetch API
* Local Storage

---

## ⚙️ How to Run the Project

1. Clone the repository:

   ```
   git clone <your-repo-link>
   ```

2. Navigate to the project folder:

   ```
   cd movie-watchlist
   ```

3. Open `index.html` in your browser

4. Add your OMDb API key inside the JavaScript file:

   ```
   const API_KEY = "your_api_key_here";
   ```
